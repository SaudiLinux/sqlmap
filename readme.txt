أداة فحص ثغرات SQL
====================

وصف الأداة
---------
أداة مكتوبة بلغة Python لفحص المواقع عن ثغرات SQL injection. تقوم الأداة بفحص الموقع المستهدف وجميع صفحاته الداخلية باستخدام مجموعة من الاختبارات المعروفة.

المتطلبات
---------
1. Python 3.6 أو أحدث
2. مكتبة requests
3. مكتبة argparse
4. مكتبة colorama للألوان
5. مكتبة beautifulsoup4 لتحليل صفحات الويب

التثبيت
-------
1. تثبيت Python من الموقع الرسمي: https://python.org
2. المكتبات المطلوبة سيتم تثبيتها تلقائياً عند تشغيل الأداة لأول مرة

طريقة الاستخدام
--------------
1. فتح Terminal أو Command Prompt
2. الانتقال إلى مجلد الأداة:
   ```
   cd مسار_المجلد
   ```
3. هناك ثلاث طرق لاستخدام الأداة:

   أ. فحص رابط واحد:
   ```
   python sql_vulnerability_scanner.py --url http://example.com
   ```

   ب. فحص مجموعة روابط من ملف:
   ```
   python sql_vulnerability_scanner.py --file url.txt
   ```
   (يجب أن يحتوي الملف على رابط واحد في كل سطر)

   ج. البحث عن الروابط في جوجل:
   ```
   python sql_vulnerability_scanner.py --file url.txt --search
   ```
   (سيقوم بالبحث عن كل رابط في جوجل وعرض النتائج)

المخرجات
--------
- عرض النتائج مباشرة في الشاشة
- حفظ النتائج تلقائياً في ملف url2.txt

ملاحظات مهمة
------------
- يجب استخدام الأداة فقط على المواقع التي تملك إذن قانوني لفحصها
- قد تحتاج لصلاحيات مدير النظام عند التثبيت على بعض الأنظمة
- في حالة وجود مشاكل في التثبيت أو التشغيل، تأكد من تثبيت جميع المتطلبات بشكل صحيح
